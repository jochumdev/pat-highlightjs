# pat-highlightjs

A [Patternslib](http://patternslib.com) pattern
which integrates [highlightjs](https://highlightjs.org/) with Patternslib.

It tells Patternslib to search for `<pre>` tags and runs `hljs.highlightBlock`
on it.

## Development

Run

    $ make
